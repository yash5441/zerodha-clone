# Zerodha Clone

A clone of the Zerodha trading platform built with React.

## Setup Instructions

1. Install Dependencies:
   ```bash
   npm install
   ```

2. Start Development Server:
   ```bash
   npm start
   ```

3. Build for Production:
   ```bash
   npm run build
   ```
